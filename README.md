# TravelWise

TravelWise is a full-stack travel and weather web application that helps users explore cities worldwide. It provides real-time weather updates, 7-day forecasts, city information, tourist attractions, hotel recommendations, and destination images through an intuitive and responsive interface.

## Features

- Search for cities worldwide
- Real-time weather information
- 7-day weather forecast
- Automatic city image slider
- City descriptions
- Top tourist attractions
- Hotel recommendations
- Google Maps integration
- Search suggestions and search history
- Responsive design
- Smooth scroll animations
- Hidden Easter Egg for Tirunelveli

## Tech Stack

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/travelwise.git
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Configure Environment Variables

Create a `.env` file inside the `server` folder.

```env
WEATHER_API_KEY=your_key
PEXELS_API_KEY=your_key
GEOAPIFY_API_KEY=your_key
```

### Run the application

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
